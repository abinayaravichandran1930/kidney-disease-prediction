# Kidney Disease Prediction Web App

A Machine Learning-based web application built using Flask to predict Chronic Kidney Disease (CKD).

## Features
- 8 clinical input parameters
- Decision Tree Classifier
- Real-time prediction
- Clean UI design

## Tech Stack
- Python
- Flask
- Scikit-learn
- HTML/CSS
- NumPy
- Pandas

## Run Locally
pip install -r requirements.txt
python app.py
